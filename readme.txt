== Description ==

Ushbu plagin WordPressda yaratilgan saytning kirill yozuvidagi o'zbek tilidagi matnini lotin yozuviga o'girib o'qishga imkon beradi. Saytingiz foydalanuvchilari uchun qo'shimcha qulaylik.

== Installation ==

1. Plagin fayllarini `/wp-content/plugins/kirilldan-lotinga` bo'limiga yuklang, yoki WordPress plagin o'rnatish sahifasi orqali saytingizga qo'shing.
2. WordPressning 'Plaginlar' sahifasini ochib, plaginni faollashtiring.
3. Saytingizning paski o'ng burchagida tugma paydo bo'ladi.
